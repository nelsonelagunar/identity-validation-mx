# Mexico Identity Validation API

## 🇲🇽 Demo Project for Mexican Market

This project demonstrates expertise in:

- ✅ Identity verification systems
- ✅ Mexican government ID validation (CURP, RFC, INE)
- ✅ Biometric facial comparison
- ✅ Digital signature (e.firma compatible)
- ✅ Bulk import processing
- ✅ Event-driven architecture
- ✅ API design (OpenAPI/Swagger)
- ✅ Kubernetes deployment

---

## ⚠️ IMPORTANT DISCLAIMER

**This is a DEMO project with SIMULATED data.**

For production use, you MUST integrate with:

| Validation | Provider |
|------------|----------|
| CURP | RENAPO (Segob) |
| RFC | SAT |
| INE/IFE | INE |
| Biometrics | Idemia, Onfido, Jumio, etc. |
| e.firma | SAT / PACs |

## 🎯 Use Cases

### 1. Fintech Onboarding
```
User Registration → CURP Validation → RFC Validation → Biometric Check
```

### 2. Digital Contract Signing
```
Document Upload → Identity Verification → Digital Signature → Audit Trail
```

### 3. Bulk Employee Verification
```
CSV Upload → Async Processing → Webhook Notification → Results Download
```
